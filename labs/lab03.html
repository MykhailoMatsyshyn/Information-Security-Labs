Шифр Playfair
та шифр Вернама

Playfair — перший практичний біграмний шифр на 5×5 квадраті. Шифр Вернама — симетричне XOR-шифрування, єдиний теоретично абсолютно стійкий шифр (OTP).

Playfair Біграмний Вернам XOR OTP
// навігація
01 Теорія
02 Playfair
03 Шифр Вернама
Короткі теоретичні відомості ▼

У цій лабораторній розглядаються два принципово різні підходи. Шифр Playfair шифрує не окремі літери, а пари (біграми) за спеціальними правилами у 5×5 таблиці — це унеможливлює простий частотний аналіз. Шифр Вернама (OTP) виконує побітовий XOR кожного символу тексту з відповідним символом ключа і є єдиним теоретично абсолютно стійким шифром за умови унікального випадкового ключа.

Шифр Playfair біграмний
[A,B] рядок → [A+1, B+1]
[A,B] стовп → [A↓, B↓]
[A,B] прямок → [ArBc, BrAc]

Пари літер шифруються за 3 правилами у квадраті 5×5. J замінюється на I. Якщо в парі однакові літери — вставляється X.

Шифр Вернама XOR / OTP
yi = xi ⊕ ki
xi = yi ⊕ ki

Побітовий XOR кожного байта тексту з відповідним байтом ключа. Симетричний: та сама операція для шифрування і розшифрування.

// біграмний шифр Playfair
Тип:Біграмна підстановка
Ключ:слово (5×5 квадрат, J≡I)
Алфавіт:англійський (25 літер)
Правила:рядок / стовпець / прямокутник
Результат з'явиться тут...
// padding X прибрано
Playfair вставляє X між однаковими літерами та в кінці — після розшифрування їх видаляють вручну.
Очищений текст:
// кроки виконання
—Натисніть «Зашифрувати»
// квадрат Playfair — шифрування біграм за 3 правилами ключ: KEYWORD
■ жовтий — вхідна біграма  ·  ■ зелений — зашифрована  ·  ■ блакитний — літери ключа
▒ зона — активне правило (рядок/стовп/прямокутник)  
// шифр Вернама (XOR / One-Time Pad)
Тип:Симетричне XOR-шифрування
Ключ:будь-який рядок (≥ довжина тексту)
Операція:y = x ⊕ k (симетрична)
Результат:hex-рядок (байти)
Результат з'явиться тут...
// кроки виконання
—Натисніть «Зашифрувати»
// XOR байт за байтом: текст ⊕ ключ = результат ключ: SECRET
■ жовтий — символ тексту (ASCII)  ·  ■ фіолетовий — символ ключа (ASCII)  ·  ■ блакитний — результат XOR (hex)
