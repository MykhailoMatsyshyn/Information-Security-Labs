Brief theory ▼

This lab covers two substitution ciphers based on a table (square). Polybius square replaces each letter with the one below in the same column. Gronsfeld cipher — numeric variant of Vigenère where shift is given by key digits.

Polybius square table substitution
Квадрат n×n, заповнений з ключем
enc: літера → літера рядком нижче
dec: літера → літера рядком вище

Keyword written first in square (no repeats), rest of alphabet added in order. EN: 5×5 (J≡I), UK: 6×6.

Gronsfeld cipher числовий Виженер
bi = (ai + di) mod n
di = digit[i mod |key|]  (0–9)

Key is decimal number. Digits cyclically assigned under text letters, each letter shifted by corresponding digit.
