ЛАБ 05 АСИМЕТРИЧНЕ ШИФРУВАННЯ

Алгоритм
шифрування RSA

RSA (Rivest–Shamir–Adleman, 1977) — перший практичний алгоритм відкритого ключа. Безпека ґрунтується на складності факторизації добутку великих простих чисел. Відкритий ключ (e, n) шифрує, закритий (d, n) розшифровує.

RSA — асиметричний алгоритм шифрування, де для шифрування та розшифрування використовуються різні ключі. Відкритий ключ можна вільно передавати, закритий залишається у власника. Основа безпеки — обчислювальна складність задачі факторизації великих чисел.

RSA — Генерація ключів асиметричний
n = p × q
φ(n) = (p−1)(q−1)
e: НСД(e, φ) = 1
d = e⁻¹ mod φ(n)

p, q — великі прості числа. Відкритий ключ (e, n) для шифрування, закритий (d, n) для розшифрування. Безпека — у складності факторизації n на множники p та q.

RSA — Шифрування блоків блочне
c = me mod n
m = cd mod n

Текст розбивається на числові блоки m < n. Кожен блок шифрується незалежно за допомогою швидкого модульного піднесення до степеня (square-and-multiply).
