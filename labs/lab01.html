Brief theory ▼

This lab covers two classes of classical ciphers: substitution ciphers (each symbol is replaced by another) and transposition ciphers (symbols are rearranged). Both are foundations of symmetric cryptography and part of modern block ciphers.

Caesar cipher simple substitution
bi = (ai + k) mod n
di = (bi − k + n) mod n

Each letter is shifted by a constant key k. Simple but easily broken by frequency analysis.

go to section ↓
Proportional substitution (Atbash) simple substitution
bi = n − 1 − ai

Alphabet maps onto itself in reverse: A↔Z, B↔Y… Self-inverse: encrypt = decrypt.

go to section ↓
Vigenère cipher polyalph. substitution
bi = (ai + ki) mod n
ki = key[i mod |key|]

Key-word is cyclically applied to plaintext. Different shift per letter prevents simple frequency analysis.

go to section ↓
Simple transposition transposition
out[key[i]−1] = block[i]
block = n symbols (|key| = n)

Text is split into equal blocks. In each block, symbols are rearranged according to the permutation key.

go to section ↓
Vertical transposition route
Write: by rows (cols = |key|)
Read: columns by sort(key)

Text is written into a rectangular table by rows. Columns are read in the order defined by the key.
