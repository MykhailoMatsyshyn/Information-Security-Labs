Шифри заміни
і перестановки

Практичне освоєння основ побудови шифрів заміни та перестановки. Вибір ключів, процедура зашифрування-розшифрування, покрокова візуалізація.

Цезар Атбаш Виженер Перестановка Вертикальна перестановка
// навігація
01 Теорія
02 Шифр Цезаря
03 Атбаш
04 Виженер
05 Перестановка
06 Вертикальна
Короткі теоретичні відомості ▼

Шифр заміни (підстановки) — метод, при якому кожен елемент початкового тексту замінюється одним або декількома знаками деякого алфавіту.

bi = (ai + k) mod n — шифр Цезаря

Шифр Виженера — багатоалфавітна заміна з ключем-словом. Кожна буква зсувається на кількість позицій, яку задає відповідна буква ключа.

bi = (ai + ki) mod n — Виженер, де ki — i-та буква ключа

Шифр перестановки — букви відкритого тексту переставляються (але не замінюються). Ключ визначає новий порядок символів.

Вертикальна перестановка — текст записується рядками в прямокутник, а зчитується по стовпцях в порядку, заданому ключем.

// шифр цезаря
Тип:Проста заміна
Ключ:зсув (0–25)
Формула:b=(a+k) mod n
Результат з'явиться тут...
// кроки виконання
—Натисніть «Зашифрувати»
// атбаш
Тип:Дзеркальна заміна
Ключ:не потрібен
Формула:b = (n−1) − a
Результат з'явиться тут...
// кроки виконання
—Натисніть «Застосувати Атбаш»
// шифр виженера
Тип:Багатоалфавітна заміна
Ключ:слово
Формула:b=(a+k) mod n
Результат з'явиться тут...
// кроки виконання
—Натисніть «Зашифрувати»
// проста перестановка
Тип:Перестановка символів
Ключ:"3 1 4 2" або "3,1,4,2"
Блок:= довжина ключа
Результат з'явиться тут...
// кроки виконання
—Натисніть «Зашифрувати»
// вертикальна перестановка
Тип:Маршрутна перестановка
Ключ:числа або слово
Запис:рядками → зчитування стовпцями
Результат з'явиться тут...
// кроки виконання
—Натисніть «Зашифрувати»
